Sapi

Quickly find the time it takes to read online articles.

Sapi looks for any online articles you're reading, then tells you how much ⏰ it will take to read based on a personalized reading test.

How will Sapi help me?

A simple issue deserves a simple solution.

1. Take a short reading test

This will help determine how fast you read, so we can calculate the minutes to read.

2. Find a web article to read

Any web article will do.

3. Sapi tells you the ⏰ it takes to read

The minutes to read will appear in the Sapi icon in Chrome's upper toolbar.

Install Sapi for free from the Chrome Web Store.
